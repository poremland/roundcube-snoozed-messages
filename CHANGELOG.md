# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-08

### Added

- Initial release of the Roundcube Snoozed Messages plugin.
- Support for one-click quick snooze options (1h, today, tomorrow, etc.).
- Support for custom date/time snooze selection.
- Automatic re-delivery of expired snoozes via background tasks.
- Secure, encrypted storage of credentials for offline restoration.
- Manual unsnooze functionality from the "Snoozed" folder.
- Comprehensive automated test suite.
- Detailed installation and contribution guidelines.
